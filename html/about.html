About

初めまして、ポートフォリオサイトを制作した「キダ マヒロ」です。

徳島出身徳島育ちの生粋の徳島アラサー男子です。

前職は、派遣社員として食品工場の製造ラインスタッフを約3年間勤めていました。製造が円滑に進められるように、環境整備に取り組んだり、製造ラインはチームプレイなので班員としっかりコミュニケーションを取り連携しながら業務にあたっていました。

趣味は、基本インドア派なので自宅で済む、ゲーム、漫画、小説、動画鑑賞、映画鑑賞、健康維持のための軽い筋力トレーニングをしています。

Skill

  • HTML5&CSS3

    HTMLは基本的な文書構造は理解できており、Class名の記述はBEM記法を意識している。CSSは基本的なことは理解しており、CSSのみでスライドショーを実装できる。

  • Java Script

    基本的なアルゴリズムの理解が乏しく、0から記述することが出来ない。ライブラリやプラグインを活用してスライドショーの実装が出来るが、理解を深めていく必要がある。

  • PHP

    PHPの記述ルール、基礎文法や外部化したファイルの読み込み方の理解はできている。PHPを利用してデータベースにデータの登録や削除、編集ができる。簡易的なアプリであれば作れる。

  • Laravelフレームワーク

    現在進行で勉強中。MVCモデル、テンプレート、レイアウト、コンポーネントについてはある程度は理解できているが、まだまだ分からないことがたくさんある。

  • MySQL

    基本的なデータの検索、挿入、削除、更新や集計、新しいDBやテーブルの作成方法を理解している。複雑なデータの操作やテーブルの結合は経験が少ないので苦手。

  • Design Soft

    それぞれのソフトの基本機能を理解している。簡単なロゴの作成、写真の加工程度なら出来る。Figmaを利用して、ワイヤーフレーム、デザインカンプの作成が出来る。
